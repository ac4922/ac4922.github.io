Task: Implement Tool Builder, Level or Map Editor
This task involves designs all map or level-related classes and may build the external tool. Since characters and items usually appear in a game map or level, the person carrying these responsibilities should work closely with the character and object implementation person to figure out the file format & saving/loading process for the game before either begins work.
