      Main Description
      Go to the home page for your group repo
      Click on “Commits” near the top of the page
      Click on the commit that contains the version of code you want us to grade
      On the top right, select Options -> Tag
      When creating the tag, enter “vX.0” as the tag name (ONLY those 4 characters where X is the current milestone number). Make sure to also include your release notes.
      After creating the tag, you should see a summary page for the tag
      Copy the URL and submit that as the link for the group milestone dropbox in myCourses.
