Task: Implement All Game States
All “minimally required” states of your game should be reachable. They don’t have to all work perfectly or look pretty, but they should all be implemented at this stage. This includes: menus, pause screens, level selection, battle modes, etc. If you have some extra states that are on your “if we have time” list, they don’t need to be implemented yet.
